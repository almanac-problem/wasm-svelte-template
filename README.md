# Svelte-Rust Offline HTML template

This repository is a template for a **very** basic application that compiles to a single, offline HTML page. It's a Sveltekit application that embeds everything inline (including WASM modules). The repo also has a Rust crate that compiles to WASM as part of the build process.

**Requirements**: JS and Rust toolchains
<br/>
**File size**: Large
<br/>
**Guarantees**: None
